# 缺血性脑卒中中医智能证据构建系统 · V1.0 前端

Next.js 单页工作台应用，按 `demo网页图` 的绿色医疗科研工作台原型实现 V1.0 专家工作流。

## 页面模式

当前 Next.js 版本保留 `v1.0前端页面.html` 的页面模式：在同一个工作台容器内通过顶部导航切换五个页面区块。

- 首页/专家工作台入口
- 标签检索与文献选择
- 结构化证据链编辑
- 证型-治法-方药-疗效指标知识图谱
- 受约束 AI 摘要生成与原文回溯

## API 配置

后端基础地址通过环境变量配置：

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

未配置或接口失败时，页面会显示最小兜底示例数据，保证演示流程不空白。

## 本地运行

```bash
npm install
npm run dev
```

构建检查：

```bash
npm run build
```

旧 Vue3/Vite 源码已备份到 `vue-backup/`。
